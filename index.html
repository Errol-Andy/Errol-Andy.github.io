SEMI-SUPERVISED CLASSIFICATION WITH GRAPH CONVOLUTIONAL NETWORKS
ICLR 2017
https://github.com/tkipf/gcn
代码版本：tf，keras，torch

定义Laplace：$L = I-D^{-\frac{1}{2}}AD^{-\frac{1}{2}}=U\Lambda U^{T}$ ，U为L的特征向量
定义Kernel：$g{\theta}(\Lambda)$
定义卷积公式 $g{\theta}*x$：$y = g{\theta}(L)x=g{\theta}(U\Lambda U^{T})x$

选择不同卷积核，卷积公式不同：

  1. $g{\theta}(\Lambda)=diag(\theta)$
    $y = g    {\theta}(U\Lambda U^{T})x=Ug_{\theta}(\Lambda)U^{T}x$

  2. $g{\theta}(\Lambda)=\sum{k}^{K}{\thetak}\Lambda^k$ ，由于$(U\Lambda U^{T})^k=U\Lambda^k U^{T}$
    $y=\sum    {k}^{K}{\thetakU\Lambda^k U^{T}x}=\sum{k}^{K}{\theta_k L^k x}$

  3. $g{\theta}(\Lambda)=\sum{k}^{K}{\theta_k}T_k(\tilde{\Lambda})$

    $y=\sum_{k}^{K}{\theta_k T_k(\tilde{L})x}$

其中$\tilde{\Lambda}=\frac{2}{\lambda{max}}\Lambda-I_N,\tilde{L}=\frac{2}{\lambda{max}}L-IN$
由Chebyshev polynomials： $T_0(x)=1, T_1(x)=x, T_k(x)=2xT{k-1}(x)-T_{k-2}(x)$

令 $\lambda_{max}=2, K=1$ 则 $ y=\theta_0 x+\theta_1 T_1(\tilde{L})x$ ，卷积公式推导如下：

Adaptive Graph Convolutional Neural Networks
AAAI 2018
https://www.jiqizhixin.com/articles/2018-01-21-8

思想：假设存在最优Laplace，为原Laplace的一个偏移。通过学习Laplace的残差来训练，加速训练过程。

优点：学习原图隐含的关系（连接新边），可变输入大小（图大小可以不同，点特征维数可以不同）；图分类效果显著

解释性：graph具有内在结构，如分子上的原子。如何找到有意义的子结构？
如苯环结构，至少需要3阶邻域，此时GCN等寻找一阶邻域的不适用。

计算两点间Generalized Mahalanobis distance，其中M对称半正定，为参数矩阵
计算Gaussian kernel，归一化得到$\hat{A}$

马氏距离：尺度无关，考虑节点间相关性

通过学习隐协方差，发掘节点间隐藏关系，作为Laplace的补充。

用于图分类：Graph Max Pooling，Graph Gather，Bilateral Filter，Batch Normalization

实验：Multi-task Prediction on Molecular Datasets，Point Cloud Object Classification

Representation Learning on Graphs with Jumping Knowledge Networks

Posted on 2019-04-01

Representation Learning on Graphs with Jumping Knowledge Networks
ICML 2018
https://zhuanlan.zhihu.com/p/40929696

思想：每层输出加权，输入最后一层。
聚合函数：线性变换；取最大值；LSTM训练每层输出，取分数最大。

优点：解决不同节点对全局和局部信息获取的不平衡问题。
缺点：效果一般；理论证明只基于简化的1阶GCN

问题背景：当前图的表示学习方法，主要是neighborhood aggregation，每次迭代都结合当前节点特征和邻域节点特征，输出新的隐层特征。

GCN缺陷：最佳两层，不能增加。对不同邻域结构的节点，输出表示来自的平滑范围不同。密度中心容易得到全局信息，边缘节点只能得到局部信息。如何找到适当邻域？

如图：当前节点居于中心，平滑范围扩散过快；当前节点居于边缘，范围扩散过慢；一旦触及中心，平滑扩散速度陡增。

本文亮点：将k层图卷积和k步随机游走联系起来

实验：GCN，GraphSAGE，GAT （谱，非谱，注意力非谱）

Deep Adaptive Image Clustering (DAC)

背景：结合表示学习和聚类。聚类难题在于预定义距离，用AE进行特征学习。缺点在于无监督学出特征表示固定后再聚类。

​ 聚类：基于距离（k-means），基于密度，基于连接（谱聚类）
​ 无监督表示学习/重构：AE，GAN。基于生成的方法不能直接得到聚类结果。
​ 成对二元性：一对样本，要么属于同类，要么属于异类，这是二值的。

目的：通过样本特征和相似性，希望每个样本得到一个one-hot的label feature。问题在于真实相似性未知。

方法：利用余弦距离衡量样本相似性，用深度卷积网络ConvNet自适应地训练参数。特征学习和聚类是一体的。

损失函数：真实类标是否同类 vs 网络学得余弦距离

余弦距离：网络学得样本的类标特征(label feature)，代表分为每一类的可能性，非负。L2归一化。样本类标特征间的点乘即为距离。

真实样本：未知。通过学习到的距离确定，设置两个阈值，希望两两异类距离大于两两同类。即选择置信度高的样本，给予伪类标。

参数更新：固定w，更新阈值函数的参数

梳理：
训练ConvNet，通过最小化log损失。损失为真实相似性和样本间余弦距离的差别。
其中，余弦距离为样本的类标特征间的；
其中，类标特征的L2范数为1，非负。最好情况为one-hot；【定理证明材料呢？】
其后，基于生成的特征，来选取参与训练的样本。
训练过程生成特征不断完善，为了逐步加入样本，lambda越来越大，两个阈值逐步靠近，相等时说明所有样本参与训练。

算法流程

  1. 随机初始化网络参数
  2. 采样batch，m个样本
  3. 根据余弦距离选择置信度高的样本，打上类标
  4. 有类标的样本系数记为1，无类标的样本系数记为0，用于求损失函数
  5. 最小化损失函数，更新网络参数
  6. 固定w，更新划分伪类标的阈值。终止条件为所有样本都参与训练。
  7. 上述迭代完成后，类标特征值最大的即为该样本类标。

实验

实验细节：阈值的选择，初始化；噪声样本【如果是图，本来就是稀疏词特征，可以加噪吗？】

评估指标：【衡量两个数据分布吻合程度】

  • RI 兰德系数： $ RI= \frac{a+b}{a+b+c+d} = \frac{a+b}{C_n^2} ​$ 范围[0,1]，越大越好；从数据选出的对中，同类分为同类，异类分为异类的概率；c、d为分错的。

  • Adjusted Rand Index (ARI)：$ RI= \frac{RI-E[RI]}{max(RI)-E[RI]} $ 范围[-1,1]，越大越符合真实分布；保证随机结果的指标接近0。

  • Normalized Mutual Information (NMI)：范围[0,1]；AMI同ARI的思想；

优势：特征具有一定可解释性

总结：最终目标是为每个样本学习一个类标特征（即归一化后每个值代表分为该类的概率），方法是学习类标特征之间的距离矩阵。由于没有真实类标，所以要利用学习结果中置信度高的创造伪类标，对这一部分有监督学习，来计算损失函数。

解释：聚类通常不构建人类用类别区分簇这种行为，故将相关类标加入表示。即为label feature。
kNN当数据量大时，分类效果好；视觉和语义具有联系。

流程：
输入数据，n个样本，每个样本t维
寻找kNN，每个近邻都对应一个label feature，d维（即最后分为d类）
计算每个近邻的权重，只和距离有关
每个label feature的值来自所有近邻在相应维度的label feature的加权
迭代更新后得到最终label feature
